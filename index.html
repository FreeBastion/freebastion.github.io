FreeBastion - A Free and Open Source Live Analysis and Routing Distro



Why FreeBastion?

FreeBastion was developed out of the need to have a secure and ready to use linux router / traffic appliance for Incident Response and Analysis. FreeBastion is not ideal for every-day routing but is ideal instead for quick diagnosis of potentially compromised or vulnerable hosts. Due to the nature of these situations, FreeBastion requires no physical storage and can be run in-place using at least two network interfaces. By using Alpine Linux we were able to accomplish this and much more!

FreeBastion fits onto a single CD and can handle not only traffic routing but also filtering. Why don't you give it a try today?

Why a new Linux Distro?

Powerful

FreeBastion is not just a router, it is also a fully feature complete monitoring and traffic isolation platform. It comes fully bundled with all the tools you may need including: snort, wireshark, ntop-ng, iptraf-ng, and much more!

Fat-free

Using only about 400MB of physical media storage, FreeBastion is lightweight and easy to use. No need to worry about unreliable internet connectivity or bandwidth limitations, you can now quickly isolate machines anywhere.

Easy to Use

Timing is critical with incident assessment so we do make it easy to setup and configure a working network isolation tool in-place with existing network infrastructure. Additionally, FreeBastion makes it easy to save settings for use later.

Open Source

FreeBastion is opensource and available to be customized and shared! All resources on our mirror are available to be used as well. Just please share resources back to us.

Frequently Asked Questions


Why a new distro?

There are plenty of good router appliances from PFSense to VyOs, however they require persistant storage and cannot easily save configuration states onto other media (they require a physical media) and are not designed for quick analysis and triage.

How do I get started?

Pull down the iso or use the bastion lbu config to get started. Then just type setup-bastion to get started and read up on LBU to save states between reboots (if that happens). Or to install to local media type setup-alpine, and follow the instructions on the Alpine Linux Getting Started Guide

Why Alpine Linux?

Alpine Linux is a very unique lightweight distribution designed for routers and other embedded systems. Unlike a traditional Linux system, it uses musl instead of glibc which lowers resource usage and attack area. Additionally, Alpine is based on rolling releases which fits our weekly release schedule.

How Often is FreeBastion Updated?

FreeBastion is rebuilt weekly to provide latest resources for users.
